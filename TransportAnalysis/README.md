# Nairobi Transport Data Analysis

## Problem
Traffic congestion affects productivity and planning in Nairobi. Understanding traffic patterns can help optimize transport systems.

## Process
- Dataset: Nairobi traffic data (synthetic or open-source).
- Tools: Python, Pandas, Seaborn, Matplotlib.
- Steps:
  1. Clean and prepare dataset.
  2. Visualize traffic volume by hour.
  3. Compare traffic across different routes.
  4. Identify peak congestion times.

## Outcome
Charts and graphs showing busiest hours and routes.

## Impact
Provides insights for city planners, ride-hailing companies, and logistics firms.
